## 빅오 표시법 (Big-O notation)
- 연산을 17N+4번 하는 솔루션 : O(17N+4) = O(N)
- 연산을 2N제곱 + 5N + 9 : O(2N제곱 + 5N +9) = O(N제곱)
- 연산을 2N로그2N번 하는 솔루션 : O(2N로그2N) = O(N로그2N)
- 연산을 k2N제곱번(k는 상수) 하는 솔루션 : O(k2N제곱) = O(2N제곱)
- 연산을 2번 하는 솔루션 : O(3) = O(1)
